##css
### clip-path
```css
// 클리핑 범위 지정
// 클리핑 범위 안의 부분 보여지고, 바깥은 숨겨짐
clip-path: circle(40%);
clip-path: polygon(50% 0, 100% 50%, 50% 100%, 0 50%);

```
### transform-origin
```css
// 원점의 기준이 되는 좌표
transform-origin: 0% 0%;
transform: rotate(45deg);
```
